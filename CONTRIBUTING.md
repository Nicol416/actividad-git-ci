# Guía para Contribuidores

## ¿Cómo contribuir?

1. Haz un fork del repositorio
2. Crea una rama con tu feature: `git checkout -b feature/mi-feature`
3. Haz commit de tus cambios: `git commit -m "feat: descripción"`
4. Haz push a tu rama: `git push origin feature/mi-feature`
5. Abre un Pull Request

## Estándares de código

- Usa mensajes de commit descriptivos
- Documenta los cambios en el CHANGELOG
